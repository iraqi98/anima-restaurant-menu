# ANIMA Café & Pizzeria 🌟

## ✨ الصفحة الرئيسية
صفحة ترحيبية تفاعلية لمقهى ANIMA مع:
- خلفية فيديو متحركة.
- زر \"View Menu\" انتقال دائري.
- أيقونات Instagram, TikTok, WhatsApp مع توهج ودوران.
- دعم AR/EN + RTL.

**رابط الواتساب**: `https://wa.me/9647890809000?text=مرحبا أود الطلب من ANIMA` (نظيف بدون إيموجي).
